The website features photographs of animals from Pixabay.com. The key to animal photography is patience. It can take a long time to get the right lighting and the best view of the animal. When photographing animals, you do not have a lot of control...you need to wait for them.

About this Website:

This website was created by David Homosena Mushegeto in 2025 as part of the Web Fundamentals course at Saint Paul College. It serves as a foundational project demonstrating core principles of modern web development, including responsive design, accessibility, and clean coding practices. Through this project, David explored essential front-end skills and began building the technical experience needed for future professional work in web development.
